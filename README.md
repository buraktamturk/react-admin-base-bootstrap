# react-admin-base-bootstrap

A Bootstrap-based admin theme package with additional UI dependencies for dashboard usage.

## Installation

```bash
npm install react-admin-base-bootstrap
```

## Package Info

- Version: `0.10.0`
- Main: `lib/esm/index.js`
- Module: `ESM`

## Peer Dependencies

- `react >=19.2.3`
- `react-intl >=8.0.11`
- `react-router >=7.11.0`

## Key Dependencies

- `@emotion/react`
- `@fortawesome/fontawesome-free`
- `bootstrap`
- `react-admin-base`
- `react-dnd`, `react-dnd-html5-backend`
- `react-password-strength-bar`, `react-responsive`, `react-select`
- `@dnd-kit/core`, `@dnd-kit/modifiers`, `@dnd-kit/sortable`, `@dnd-kit/utilities`
- `sweetalert2`, `nprogress`, `prettysize`, `file-dialog`, `modal-cropper`

## Scripts

- `npm run build`: `tsc`
- `npm run watch`: `nodemon -e js,jsx,ts,tsx --exec npm run build`

## Release Hint

Make sure the installed `react-admin-base` version is compatible with this package before publishing.
